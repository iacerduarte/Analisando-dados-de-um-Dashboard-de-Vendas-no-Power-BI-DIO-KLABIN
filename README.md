# Analisando-dados-de-um-Dashboard-de-Vendas-no-Power-BI-DIO-KLABIN

📊 Power BI – Relatório de Vendas e Análise de Lucro
📌 Descrição do Projeto

Este projeto tem como objetivo replicar e expandir um relatório desenvolvido durante o curso de análise de dados utilizando o Power BI.

Foram replicadas duas páginas do relatório original e criada uma terceira página personalizada, com o propósito de desenvolver habilidades na construção de visualizações e análise de dados.

O desafio tem como foco principal a prática na criação de visuais, organização do layout do relatório e apresentação de métricas relevantes.

📂 Fonte de Dados

Os dados utilizados neste projeto estão disponíveis no repositório original: 🔗 https://github.com/julianazanelatto/power_bi_analyst

📈 Estrutura do Relatório

O relatório é composto por três páginas principais.

1️⃣ Página 1

Replicação do relatório apresentado durante o curso.

Contém visuais de análise relacionados a:

Vendas

Lucro

Segmentos de mercado

Distribuição de dados

2️⃣ Página 2

Continuação da replicação do relatório original.

Apresenta análises complementares para exploração dos dados e compreensão dos indicadores de negócio.

3️⃣ Página 3 – Página criada no desafio

A terceira página foi desenvolvida de forma independente e contém os seguintes visuais:

🗺️ Mapa 1

Soma de Sales e Unidades Vendidas por País

Objetivo:

Visualizar a distribuição geográfica das vendas

Comparar volume de vendas entre países

🗺️ Mapa 2

Soma de Lucro (Profit) por País

Objetivo:

Identificar quais países apresentam maior geração de lucro

Comparar desempenho financeiro por região

🥧 Gráfico de Pizza

Distribuição de Lucro por Segmento

Segmentos analisados:

Government

Small Business

Channel Partners

Midmarket

Enterprise

Objetivo:

Identificar quais segmentos contribuem mais para o lucro total.

🧩 Melhorias Aplicadas

Durante o desenvolvimento do relatório foram realizadas algumas melhorias importantes:

Organização e melhor disposição dos visuais no relatório

Renomeação dos visuais para títulos mais claros e objetivos

Estruturação do relatório para melhor interpretação dos dados
